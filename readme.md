# Tailwind Compositor Starter Static

Starter kit for [tailwind-compositor](https://github.com/a7sc11u/tailwind-compositor).

Uses, parcel, Postcss, Tailwind and a minimal theme.

### Install

```
  yarn install
  // or
  npm install
```

### Run

```
  yarn develop
  // or
  npm develop
```

### Build

```
  yarn build
  // or
  npm build
```
